Gene Properties

Filter by gene properties. For gene position, enter the lower bound in the first input field and the upper bound in the second input field. Then, select the checkbox. For example, if I wanted to search for genes between 300,000bp and 600,000bp, I would enter 300000 and 600000 respectively.
For Operon, if you only want to know which genes are controlled by annotated operons, simply check the box. If you want to see which genes are controlled by the same operon as another gene, enter the NCBI locus tag in the specify input form. For example, if I want to know which genes share an operon with glyceraldehyde-3-phosphate dehydrogenase CPn0624, I would enter CPn0624 in the input field and tick the checkbox.

Ortholog Cliques

Filter by ortholog cliques. Use this option to filter by genes that form orthologous cliques with the selected organisms. For example, if I wanted all genes in Cmur and L2, I would check the box next to enable ortholog search, then select C. trachomatis 434/BU and then select C. muridarum Nigg. If I wanted to search for genes that did not have any orthologs, I would only check enable ortholog search and leave the organisms checkboxes blank.
